"Testing programs by means of jMeter"

Mentor: Lev Gorfel | 2021

Program performance testing and load testing are the most important skills of a modern engineer. The course will help you get acquainted with general load testing methodologies and teach you how to test the performance of http queries using the apache benchmark and jmeter tools. the course consists of real examples and includes three independent tasks with the subsequent decision. The lectures will be of interest to software validation engineers and web application developers.
